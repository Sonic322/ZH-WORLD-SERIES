ZHLADDER

ZHLADDER is one of the most commonly used places to find 1v1 and 2v2 ranked games.

They are using the ELO rating system with some improvements based on the win rates of the armies in the Generals Zero Hour.

These have a ranked format where each player has a rating based on who they play, whether they win, and with which armies they win.

You must register in the server with a username and then play games to get a rank! If you lose, you must report it in the server with the name of the opponent and the armies each player had.

Qualification will use the monthly leaderboard found on the Discord. If you have any questions on this, feel free to reach out to GameReplays staff, or Sonic.

A brief VIDEO tutorial on how to play on ZHLADDER can be found below:

    If you're too lazy to watch the video but want to start playing on the ZHLADDER, here's all you need to do:
  • 1. Join the Discord serverif you’re not there yet.
  • 2. Register using the /reg command in any 1v1 lobby channel.
  • 3. Find an opponent you'd like to play a ladder game with.
  • 4. After the match, if you lost, report your loss using the /m command.

If you have any questions, check the "Info" channel or ask members in the lobby - they'll help you.
